Product Manufacturing (MRP) Dashboard

Dashboard for Product Manufactuing in Odoo

Product Production Dashboard in Odoo

This app adds a Product Manufacturing / Production dashboard.

Main Features:

  • This module will allow your users to view the Product MRP Dashboard.
  • You can see Product Image, Name on Dashboard.
  • You can see Productions Button on Dashboard and while clicking on Productions Button which shows Product Productions (Manufacturing Orders).
  • You can see the Work Orders Button on the Dashboard and while clicking Work Orders Button which shows Product Work Orders.
  • You can see BOM Button on Dashboard and while clicking BOM Button which shows Product BOM.
  • You can see Unbuild Orders Button on Dashboard and while clicking Unbuild Orders Button which shows Product Unbuild Orders.
  • You can see Scrap Orders Button on Dashboard and while clicking Scrap Orders Button which shows Product Scrap Orders.
  • You can see Routing Button on Dashboard and while clicking Routing Button which shows Product Routing.
  • For more details please check below screenshots and watch the video.
